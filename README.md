## 注册接口

```
POST /add
```

请求参数

```
{
	"username": "用户名",
	"password": "密码",
	"sex": "性别",
	"age": "阿拉伯数字",
	"address": "居住地"
}
```

**响应参数**

```
{
	"status": 200
	"msg": "注册成功"
}
```

**失败时：**

`HTTP/1.1 400 Bad Request`

```
1. username参数已存在
2. 缺少参数 username
3. 缺少参数 password
4. 缺少参数 sex
5. 缺少参数 age
6. 缺少参数 address
```
